# Asking the Auditor

This repository contains the code used in the visiting scholarship at the Dutch Court of Audit. The goal of the scholarship was to develop new methods to study the Court's impact on politics and the public debate. The project culminated in three subprojects:

1. Semantic footprints: I used sentence embeddings to link sentences found in the Court's reports to ones found in parliamentary debate.
2. Topical Sediments: I used a combination of topic models and references to the Court to reconstruct the long-term discursive development of performance auditing.
3. Procedural interactions: I used computationally classified parliamentary written questions to the Court (and the Court's answers) to study interactions between these two institutions.
